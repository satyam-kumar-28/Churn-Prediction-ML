## 🚀 Day 1: Data Loading

- Checked structure using `.info()`, `.shape()`
- Viewed sample data using `.head()`

## 🚀 Day 2: Data Cleaning
- Converted `TotalCharges` from string to numeric
- Handled missing values
- Converted categorical data (Yes/No → 1/0)
